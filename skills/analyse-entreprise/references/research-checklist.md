# Checklist de recherche — Analyse Entreprise

## A. Sources officielles
- [ ] Site corporate
- [ ] Page investisseurs / rapports annuels
- [ ] Communiqués de presse
- [ ] Conditions commerciales / tarification

## B. Indicateurs business
- [ ] Produits/services principaux
- [ ] Segments clients
- [ ] Différenciation concurrentielle
- [ ] Partenariats

## C. Indicateurs économiques
- [ ] Chiffre d'affaires
- [ ] Croissance YoY
- [ ] Marge / rentabilité
- [ ] Financements / dette
- [ ] Signaux de stabilité ou tension

## D. Indicateurs digitaux
- [ ] Qualité du site et parcours utilisateur
- [ ] Présence SEO
- [ ] Réseaux sociaux (activité, engagement)
- [ ] Acquisition digitale (ads/newsletter/contenu)
- [ ] Niveau d'automatisation et data

## E. Concurrence
- [ ] 3-5 concurrents comparables
- [ ] Positionnement prix/valeur
- [ ] Forces/faiblesses relatives

## F. Contacts publics
- [ ] Email entreprise
- [ ] Téléphone entreprise
- [ ] Adresse / siège
- [ ] Liens sociaux officiels validés

## G. Validation
- [ ] Chaque donnée critique a une source
- [ ] Dates des sources explicites
- [ ] Aucune inférence présentée comme fait
